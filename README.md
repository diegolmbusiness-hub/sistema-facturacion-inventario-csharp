# Sistema de Inventario y Facturación 🛒

Sistema desarrollado en C# para gestionar inventario y facturación en una tienda de ropa.

## 📌 Descripción
Proyecto grupal universitario que permite registrar, actualizar y consultar productos, clientes, proveedores, compras y ventas. Implementa arquitectura en tres capas para un mejor manejo del sistema.

⚠️ Nota: El proyecto aún necesita algunos ajustes y mejoras.

## 🔐 Credenciales de acceso

## 🛠️ Tecnologías utilizadas
- Lenguaje: C#
- Entorno: Visual Studio 2019
- Base de datos: SQL Server (LocalDB)
- Arquitectura: 3 capas (Datos, Negocio, Presentación)

## 🧱 Arquitectura
- **Capa de Datos:** Manejo de la base de datos y procedimientos almacenados.
- **Capa de Negocio:** Lógica del sistema.
- **Capa de Presentación:** Interfaz gráfica para el usuario.
